# 🧠 SECOND BRAIN / Mission Control

> AI Agent Enterprise Dashboard – inspiriert von OpenClaw Mission Control

## Features

| Tab | Beschreibung |
|---|---|
| 🎯 Mission Control | Live-Metriken, aktive Agenten, Aktivitäts-Feed |
| ✅ Tasks | Aufgaben-Queue mit Filtern & Quick-Add |
| 💬 Chat | Direkt-Chat mit jedem Agenten |
| 🏢 Org | Org-Chart des gesamten AI-Teams |
| 🏠 Office | Pixel-Art Büro mit Echtzeit-Agen­ten­stand |
| 🧠 Memory | Identitäten, Regeln & Prompts je Agent |

## Starten

```bash
# Option A – Updater (empfohlen)
updater.bat

# Option B – Manuell
npx serve . -p 4242
# → http://localhost:4242
```

## Anforderungen
- Node.js 18+
- Moderner Browser (Chrome, Edge, Firefox)
