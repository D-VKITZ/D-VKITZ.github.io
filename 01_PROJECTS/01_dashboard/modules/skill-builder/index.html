Was ist der Skill Builder?

Hier kombinierst du einzelne Actions zu wiederverwendbaren Skills — Bausteine fuer Workflows und Agenten.

🔗 Builder-Kette:

Action - Skill - Workflow - Agent - Team

So gehts:

  1. Ziehe Actions von links auf die Flaeche
  2. Benenne deinen Skill und whle eine Kategorie
  3. Exportiere oder sende direkt an Workflow/Agent Builder

💡 Tipp: Actions knnen aus dem Action Builder importiert werden!
