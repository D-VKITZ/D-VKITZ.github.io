Agent Pool

👥 Team Builder — Komplett-Guide

Koordiniere mehrere Agenten als Team nach BMAD-Methodik.

Builder-Kette:

Action → Skill → Workflow → Agent → Team → Leadership

Features

  • Agent-Pool: Alle Agenten aus der DkZ-Datenbank, filterbar nach Tags und durchsuchbar
  • BMAD Templates: v00 = Original (7 Kern-Rollen), plus konfigurierte Varianten nach Genre
  • Rollen & Tokens: Detaillierte Konfiguration pro Agent (Leader/Worker/Reviewer/Validator/Specialist)
  • Loop-Funktion: Stateless (jede Runde frisch) oder Memory (Gedächtnis zwischen Runden)
  • Mini-Playbook: Definiere Schritte für den Team-Durchlauf
  • Kostenrechner: Live-Berechnung basierend auf Token-Budget und Modell-Raten
  • Demo-Lauf: Visuell animierte Simulation des Team-Durchlaufs
  • Export: Team als JSON exportieren oder lokal speichern

BMAD-Methodik

BMAD (Breakthrough Method for Agile AI-Driven Development) glaubt: Ein Agent kann nicht alles gleichzeitig sein. Spezialisierung eliminiert Halluzinationen und steigert Qualität.

v00 Original Rollen: James™ (Master), PM, Architekt, Developer, Reviewer, Tester, Dokumentar

Konfigurierte Varianten werden automatisch kategorisiert mit Genre-Tags und Technik-Symbolen.
