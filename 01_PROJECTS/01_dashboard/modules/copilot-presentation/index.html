🏗️ Builder Tab

Tutorial

🏗️ Builder — Schritt für Schritt

🤖 Button klicken
Unten rechts auf den violetten 🤖 Button klicken → Copilot-Panel öffnet sich.
🏗️ Builder Tab wählen
Im Panel oben den "🏗️ Builder" Tab klicken → Builder-Oberfläche erscheint.
📄 Dokumente hochladen (optional)
"📁 Datei wählen" für einzelne Dateien (TXT, MD, JSON, HTML, CSS, JS, PDF) oder "📂 Ordner wählen" für komplette Verzeichnisse. Die Dateien werden als Kontext an den LLM gesendet.
📌 Lokalen Pfad angeben (optional)
Pfad eintragen, z.B. C:\DEVKiTZ\01_PROJECTS\ — wird als Kontext-Info an den Prompt angehängt.
🔧 Optionen wählen
Checkboxen: DkZ™ Design, Shared Scripts, Glassmorphism, Responsive, Dark + Light, Offline-First. Standardmäßig sind die ersten 4 aktiviert.
🎨 Branding konfigurieren
DEVKiTZ™ Logo ist standardmäßig AN (Checkbox). Abwählbar. Alternativ eigenes Logo hochladen → wird als Base64 in localStorage gespeichert und beim Build eingebettet.
🚀 Build starten
Klick auf den roten "🚀 Build starten" Button → Alle Optionen werden zu einem strukturierten Prompt kompiliert und automatisch an den Chat gesendet. Der aktive LLM generiert das Ergebnis.
