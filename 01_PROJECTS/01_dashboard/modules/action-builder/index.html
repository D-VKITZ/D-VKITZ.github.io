Was ist der Action Builder?

Hier definierst du einzelne Aktionen — die kleinsten Bausteine deiner Automations-Kette.

🔗 Builder-Kette:

Action - Skill - Workflow - Agent - Team

So geht's:

  1. Ziehe eine Aktion von links auf die Flaeche
  2. Konfiguriere sie (Name, Timeout, Retry)
  3. Exportiere als JSON/YAML oder sende an den Skill Builder

Shortcuts:

Esc Schliessen   / Suche   Ctrl+S Speichern
