# 📝 Notes-Manager

> **Modul:** MOD-071 | **Version:** v0.01 | **Status:** Planned
> **Kategorie:** Productivity

## Beschreibung
4-Ordner Notiz-System mit Inbox, Aktiv, Archiv (short/long), Meeting. Push-to-Talk Voice, 30-Tage Auto-Archiv, DuckDB Logging, Google Sheets Sync.

## Features (8)
- 📂 4-Ordner Struktur (Inbox → Aktiv → Archiv → Meeting)
- 🏷️ 3 Präfixe: P-Projekt, I-Idee, R-Referenz
- 🎤 Push-to-Talk Meeting-Aufnahme
- 🗣️ Voice Commands (Meeting starten/beenden)
- 📦 30-Tage Auto-Archiv
- 🗄️ DuckDB Logging
- 📊 Google Sheets Sync (15 Min)
- 💾 JSON Local Storage (Offline-First)
