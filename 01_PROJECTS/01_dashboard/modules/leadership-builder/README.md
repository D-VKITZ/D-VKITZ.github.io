## Was ist Leadership?

Ein Schritt weiter als Teams: **Leaderships** sind Meta-Orchestratoren, die mehrere Teams koordinieren.  
Ein Team-Leiter analysiert Situationen, verwaltet Informationen und delegiert fallbasiert an das richtige Team.

## Builder-Kette

```
Action → Skill → Workflow → Agent → Team → Leadership
```

## Features

- 🏢 Team-Leiter erstellen (Name, Icon, System-Prompt, Token-Budget, Model)
- 👥 Multi-Team Zuweisungen (aus gespeicherten Teams oder BMAD-Templates)
- 🔀 Routing-Regeln (Bedingung → Team)
- 📋 CTO / CMO / CISO Templates
- 🔄 Loop-Funktion (Stateless / Memory mit konfigurierbaren Feldern)
- 💰 Aggregierter Kostenrechner (Leader + alle Teams)
- ▶️ Demo-Lauf mit visueller Animation
- 📊 Organigramm-Visualisierung
- ℹ️ Info-Guides bei jeder Sektion
- 💾 Speichern & Exportieren (JSON)
