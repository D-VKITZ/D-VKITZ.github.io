⚡ NLM Content-Pipeline — 6 Phasen zum perfekten Output

🔧 IMMER zuerst ausführen (Session-Setup):

# PowerShell — PATH + Encoding
$env:Path = "C:\Users\BAZE²\.local\bin;$env:Path"
$env:PYTHONIOENCODING = "utf-8"

# Auth prüfen
nlm notebook list
1 Notebook erstellen Start 
nlm notebook create "DkZ - [THEMA]"
# → Gibt Notebook-ID zurück
2 Quellen hochladen Sources 
nlm source add NB_ID --file "DATEI.md" --wait
nlm source add NB_ID --url "https://..." --wait
nlm source add NB_ID --youtube "https://..."
3 Content generieren Create

✅ Pflicht (Minimum 3):

nlm report create NB_ID --confirm
nlm slides create NB_ID --confirm
nlm audio create NB_ID --language de --confirm

⚡ Optional:

nlm video create NB_ID --confirm
nlm mindmap create NB_ID --confirm
nlm infographic create NB_ID --confirm
nlm data-table create NB_ID --description "..." --confirm
4 Status prüfen Wait 
nlm studio status NB_ID
# Warten bis alle "completed" sind
5 Download Save 
nlm download report      NB_ID ART_ID --output report.md
nlm download slide-deck  NB_ID ART_ID --output slides.pdf
nlm download audio       NB_ID ART_ID --output podcast.wav
nlm download mind-map    NB_ID ART_ID --output mindmap.json
nlm download video       NB_ID ART_ID --output video.mp4
nlm download data-table  NB_ID ART_ID --output data.csv
