# 📚 Wiki Viewer

> **Modul:** MOD-073 | **Version:** v1.0.0 | **Status:** Active
> **Kategorie:** System

## Beschreibung
DEVKiTZ™ Wiki Viewer — Interaktiver Markdown-Archiv-Browser mit Suche, TTS, Multi-Select, ZIP-Download und Mobile Version.

## Features (12)
- 📝 Vollständiger Markdown Renderer (Tabellen, Code, Listen, Task-Listen)
- 🔍 Global Search mit Echtzeit-Highlight über alle Seiten
- ↑↓ Sort by Name / Dateigröße
- 🔊 TTS Vorlesen (Deutsch, Web Speech API)
- ☑ Multi-Select mit Selektions-Leiste
- 📦 ZIP-Download (JSZip) — alle oder gewählte Seiten
- 📋 Clipboard Copy — Markdown-Quelltext
- ⌨️ Tastenkürzel: ←/→ Nav, `/` Suche, `Esc` Stop
- 📊 Stats: Seiten, Wörter, KB
- 📱 Mobile Version (mobile.html)
- 🎨 DkZ™ Design System v2
