# 🔄 Workflow Viewer

> **Modul:** MOD-072 | **Version:** v2.0 | **Status:** Active
> **Kategorie:** Monitor

## Beschreibung
Live Workflow Monitor im n8n-Style mit Node Canvas, SVG Bezier Connectors, animierten Flow-Partikeln und 4 View-Modi.

## Features (12)
- 🔀 n8n-Style Node Canvas mit SVG Connectors
- 📊 4 View Modi: Workflow / Agent / Team / Loop
- 🏗️ 12 Builder Workflows animiert
- ⚡ Live Simulation Tick (3.5s)
- 🤖 6 Agent Cards (Karma, Uptime, LLM)
- 👥 5 Team-Matrix mit Status-Ampel
- 🔄 8 Loop Monitor mit Ring-Animation
- 📜 Live Log Feed (INFO/OK/WARN/ERR)
- ⏸️ Quick Actions: Pause, Restart, Abort
- 💫 Node Pulse + Shake Animationen
- 🎨 DkZ™ Design System v2
- 📋 Log Export Funktion
