📋 WordPress Installation (SSH)

1
WordPress herunterladen
cd public_html && wget https://wordpress.org/latest.tar.gz
2
Entpacken & aufräumen
tar -xzf latest.tar.gz && mv wordpress/* . && rmdir wordpress && rm latest.tar.gz
3
Config erstellen
cp wp-config-sample.php wp-config.php
DB-Daten im Hostinger hPanel → MySQL Databases
4
Config bearbeiten
nano wp-config.php
DB_NAME, DB_USER, DB_PASSWORD, DB_HOST eintragen
5
Berechtigungen setzen
chmod 755 -R . && chmod 644 wp-config.php
6
Im Browser installieren
https://deine-domain.de/wp-admin/install.php öffnen
