# Builder Install Skill

Dieser Skill wird vom Builder-Agenten genutzt, um externe Ressourcen in das DEVKiTZ Oekosystem zu integrieren.

## Eiserne Regel
**Aendere so viel wie noetig, aber so wenig wie moeglich.**

## Ablauf
1. **Analysiere Link:** Lese das GitHub Repo oder das YouTube Video (via r.jina.ai) aus.
2. **Mapping:** Vergleiche die externe Architektur mit unserem DkZ-Regelwerk (REGELWERK.md).
3. **Integration:** Implementiere den Code oder die Konzepte in unser System.
4. **Wissenssicherung:** Schreibe die Learnings in ein Markdown-Walkthrough ins Obsidian Second Brain.
