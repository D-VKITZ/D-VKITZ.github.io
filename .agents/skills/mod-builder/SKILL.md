# Mod Builder

Automatisierter Generator für neue DkZ Dashboard Module.

## Generiert

1. `index.html` mit DkZ Design System v2
2. `features.json` mit Feature-Status
3. Hub-Registry Eintrag
4. Shared Scripts Integration
5. Onboarding Modal

## Nutzung

```
/create-module
```

Folge dem interaktiven Workflow für Modul-Name, Kategorie und Features.
