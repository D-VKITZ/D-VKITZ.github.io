# DkZ Skillpack

Kuratierte Sammlung externer Skills, integriert ins DEVKiTZ™ Ökosystem.

## Enthaltene Pakete

| Paket | Beschreibung |
|:------|:-------------|
| graphify | Codebase-Analyse und Zusammenhänge visualisieren |
| get-shit-done | Fokussiertes Arbeiten ohne Ablenkung |
| ui-ux-pro-max | Premium UI/UX Design-Patterns |
| everything-claude-code | Alle Claude Code Best Practices |
| Homio-main | Smart Home Automatisierung |
| gremio | Team-Koordination und Workflows |
| awesome-cli-apps-master | CLI App Referenzen |
| llm-wiki-template-main | LLM Wiki Templates |

## Nutzung

Jedes Paket hat eigene Unterordner mit spezifischen SKILL.md Dateien.
